This campaign relies on you to spread it.

We've created the materials, now just download the files and put your details on it (or not) and spread it however you can.

weQUIT Smoking is designed to coincide with WorldWide No Tobacco Day on May 31st.

How it works

The best way to tell people about how you quit smoking is to tell them face to face. So these materials are designed to have your contact details attached.
However, sometimes that is not possible so we have the website weQUIT.com.au people can visit instead.

Website banners

If you own a website or are able to use these web banners please do so. All banners use alpha transparent pngs which will look bad in ie6. Feel free to request a size that suits you better
